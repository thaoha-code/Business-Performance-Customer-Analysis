# E-commerce Business Performance & Customer Sentiment Analysis

## Project Overview
This project focuses on analyzing the business performance of an e-commerce platform and understanding customer feedback through data analysis and machine learning. The project aims to support data-driven decision making by evaluating sales performance, customer purchasing behavior, shipping efficiency, and customer sentiment from reviews.

Data is processed using Python and visualized using interactive Tableau dashboards to provide clear and actionable business insights.

---

## Dataset
The project uses multiple raw CSV files related to an e-commerce system, including:

- blinkit_orders.csv  
- blinkit_order_items.csv  
- blinkit_customers.csv  
- blinkit_products.csv  
- blinkit_customer_feedback.csv  
- blinkit_delivery_performance.csv  
- blinkit_marketing_performance.csv  
- blinkit_inventory.csv  
- blinkit_inventoryNew.csv  

All raw datasets are stored in the `data` folder and are used for academic analysis purposes.
